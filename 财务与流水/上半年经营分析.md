# 馋唻买听涛路店 2026年2-6月经营分析

## 一、整体结论

| 结论项 | 判断 |
|---|---|
| 总体表现 | 店铺有销售能力，但净利润偏薄，抗风险能力较弱。 |
| 收入趋势 | 3月达到高峰，4月后明显回落，5-6月稳定在4万元左右。 |
| 利润情况 | 2-6月合计净利润10,111.89元，整体净利率约4.1%。 |
| 核心问题 | 收入规模不足以充分覆盖工资、房租、水电等固定支出。 |
| 积极信号 | 5-6月食材成本率明显下降，说明后期成本控制有所改善。 |
| 经营重点 | 优先把月收入恢复到5万元以上，同时保持食材成本率在45%-48%左右。 |

## 七、主要问题与原因

| 问题 | 数据表现 | 可能原因 | 影响 |
|---|---|---|---|
| 3月后收入明显下滑 | 收入从3月73,829.05降到6月40,085.60。 | 平台流量下降、活动减少、竞品影响、复购不足。 | 固定成本无法摊薄，利润快速变薄。 |
| 淘宝外卖下滑较大 | 淘宝从3月28,180.51降到5月10,340.77、6月11,764.03。 | 曝光、排名、活动、价格或转化率出现问题。 | 直接拖累总收入。 |
| 京东外卖逐月走低 | 京东从3月9,171.69降到6月4,236.47。 | 新渠道热度下降，运营动作不足。 | 新增渠道没有持续放量。 |
| 固定支出压力大 | 6月工资、房租、水电合计19,333，占收入48.2%。 | 收入规模不足，人力和固定费用难以下降。 | 毛利被固定支出吃掉。 |
| 推广效果不清晰 | 6月推广1,342.40，但收入低于5月。 | 缺少按平台、按活动的投产比核算。 | 容易花钱维持流水而非创造利润。 |
| 堂食和私域偏弱 | 微信堂食从3月8,847.66降到6月5,501.92。 | 到店复购、社群、附近客群运营不足。 | 过度依赖外卖平台，利润质量受限。 |

## 八、经营建议

| 优先级 | 建议 | 具体做法 | 目标 |
|---|---|---|---|
| 高 | 把月收入恢复到5万元以上 | 重点恢复淘宝、京东，同时稳住美团主力渠道。 | 先脱离盈亏平衡线，恢复基本利润。 |
| 高 | 单独复盘淘宝外卖 | 查看曝光、访问、下单、评分、活动、配送范围、主推品价格。 | 找出3月后下滑原因。 |
| 高 | 稳住美团外卖 | 保持评分、出餐速度、爆款套餐、活动节奏和店铺排名。 | 守住当前最稳定收入来源。 |
| 高 | 控制工资占比 | 低峰减少人手，高峰集中排班，工资占收入尽量控制在22%-25%。 | 降低固定成本对利润的压迫。 |
| 中 | 保持食材成本率45%-48% | 做周盘点，减少滞销备货，提高高毛利产品占比。 | 保住5-6月已经改善的毛利水平。 |
| 中 | 建立平台利润表 | 每个平台记录销售额、扣点、推广费、退款、实收、估算食材成本。 | 判断哪个平台真正赚钱。 |
| 中 | 推广设置止损线 | 推广费超过平台销售额3%-5%时必须评估效果，连续无增长则暂停。 | 避免无效推广吞掉利润。 |
| 中 | 增强堂食和私域复购 | 到店加微信、老客券、工作日午餐套餐、附近小区/办公点定向优惠。 | 提高无平台扣点的优质收入。 |
| 低 | 修正表格公式 | 4月渠道合计少计17.55元，疑似抖音来客未纳入合计。 | 避免后续统计口径错误。 |

## 九、下月经营目标建议

| 目标项 | 建议目标 | 原因 |
|---|---:|---|
| 月收入 | 50,000元以上 | 当前4万元左右基本只是盈亏平衡。 |
| 净利润 | 5,000元以上 | 低于这个水平，经营回报偏弱。 |
| 食材成本率 | 45%-48% | 5-6月已接近该水平，应继续稳定。 |
| 工资占收入 | 22%-25%以内 | 6月接近30%，压力偏大。 |
| 推广费占收入 | 3%-5%以内 | 超过后必须验证是否带来新增订单。 |
| 堂食及私域收入 | 8,000元以上 | 降低对平台流量的依赖。 |

## 十、最终判断

| 判断 | 内容 |
|---|---|
| 现在的问题 | 不是完全没有销售，而是收入规模下降到刚好覆盖成本的位置。 |
| 最关键动作 | 把月收入从4万元拉回5万元以上。 |
| 最需要保住的优势 | 5-6月食材成本控制已经改善，需要继续保持。 |
| 最需要警惕的风险 | 固定成本高、平台收入下滑、推广投产比不清晰。 |
| 经营方向 | 稳美团、救淘宝和京东、做堂食私域、控工资占比、按平台核算真实利润。 |
