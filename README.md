# 馋味买运营中心

以福鼎肉丸为核心产品，围绕“主外卖、副堂食”的策略，首店落地杭州。该仓库用于记录品牌从0到1的全过程，包括品牌建设、开店流程、运营管理、财务流水、法务与资产、招商加盟等，形成标准化与数据化的沉淀，为未来的复制扩张与招商提供依据。

- 首店面积：约50㎡
- 年租金：约7.5万
- 经营模式：主外卖 / 副堂食
- 预计开店成本：约12.5万
- 供应链：已稳定

---

## 目录结构

- 品牌档案/
  - 记录品牌故事、愿景定位与品牌视觉规范
- 开店流程/
  - 从选址租约、装修设计、证照办理到试营业的完整流程
- 运营管理/
  - SOP标准、人员培训、供应链菜品、外卖平台、堂食运营等
- 财务与流水/
  - 成本预算、开店成本、日常流水与利润报表
- 法务与资产/
  - 商标与知识产权、合同协议、资产台账、合规与安全
- 招商加盟/
  - 加盟政策、流程、手册与审核评估机制
- 市场与品牌/
  - 年度营销计划、社媒与公关、物料模板
- 数据与分析/
  - 指标体系、数据看板设计与月度复盘
- 店铺档案/
  - 首店的详细档案：信息、装修记录、运营日志、流水台账
- 模板与示例/
  - 常用文档与表格模板，统一格式便于复制与加盟
- 工具与脚本/
  - 数据处理脚本或自动化工具（后续补充）

---

## 使用建议

- 版本管理：按阶段或月份提交，备注清晰（如“2026-01 首店装修阶段总结”）
- 标准化：所有流程尽量形成SOP与模板，避免口口相传
- 数据化：核心指标稳定采集，沉淀报表与看板
- 可复制：目录与文档保持简洁统一，便于加盟商快速上手
