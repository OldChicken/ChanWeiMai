# 品牌与市场

整合“品牌档案”与“市场与品牌”，统一沉淀品牌基础资产与传播策略，减少目录分散、提升复用效率。

## 一、品牌基础
- 品牌故事：起源、理念、差异化（福鼎肉丸的工艺与风味）
- 愿景与定位：价格带、客群画像、区域拓展策略
- 品牌视觉：Logo、色彩、字体、应用规范（门头/海报/外卖主图/菜单）

## 二、市场与传播
- 营销计划：年度/季度规划、预算分配、渠道策略
- 社媒与公关：内容日历、投放计划、KOL合作与社区共创
- 物料模板：门头、海报、菜单、外卖主图与详情页规范（统一版式）

## 三、素材与模板管理
- 素材库：设计源文件、导出规范、尺寸与格式要求
- 版本管理：命名规则与版本号（如：YYYYMM-主题-渠道）
- 复用清单：可复用的文案、海报、套餐图与活动元素

建议：统一视觉模板与素材库，确保所有渠道一致呈现；通过版本管理提升复用与协同效率。
