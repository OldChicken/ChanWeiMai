# 品牌色与元素（暂定）

依据当前门头设计效果图初步提取，后续以最终VI与源文件为准。

## 颜色建议（近似值，待校准）
- 主色（橙）：#F56B1C / #FF6A00（门头主体与背景）
- 辅色（绿）：#2FA64A / #169C3A（门头下沿与角标）
- 强调（红）：#E6452E（文字强调与视觉焦点）
- 中性色：白 #FFFFFF；深灰 #333333（文字与背景对比）

说明：以上为近似取值，待收到原始设计稿或色卡后统一校准；不同平台对色彩压缩与显示有差异，导出时注意对比与可读性。

## 元素建议
- 吉祥物/角标：左上或右上角统一位置，尺寸稳定，避免遮挡主体内容。
- 门头字形：粗壮、识别度高的标题字；在线物料保持同风格但不必完全一致。
- 版式风格：高对比、信息简明（主图≤20字）；主副色搭配突出层级。

## 使用范围
- 外卖主图与详情页、海报、菜单、卡券等线上物料。
- 线下物料（门头、贴纸、POP）参考同一色系与版式原则。
