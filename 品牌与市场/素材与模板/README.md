# 品牌与市场｜素材与模板

用于集中管理品牌视觉与市场物料模板，提升统一性与复用效率。

- 视觉规范模板：Logo使用、色彩与字体、版式规范、导出与尺寸要求
- 物料模板索引：门头、海报、菜单、外卖主图与详情页、卡券与话术
- 主图与详情页规范：见 主图与详情页规范.md（尺寸/版式/命名/审核清单）
 - 品牌色与元素（暂定）：见 品牌色与元素（暂定）.md（近似色值与元素建议，待校准）
 - 门头设计入口：见 门头设计/README.md（放入图片后可在页内直接预览）
- 命名与版本管理：YYYYMM-主题-渠道（示例：202601-冬季促销-美团）

建议：所有品牌物料类模板在此维护，通用运营/财务类模板仍放置在“模板与示例”。
