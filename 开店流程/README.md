# 开店流程

- 项目概览：时间线与里程碑（选址→签约→证照→设计→施工→试营业→正式营业）
- 选址与租约：商圈评估、客流、租约条款、物业配套
- 证照办理：营业执照、食品经营许可、消防与健康证
- 装修设计：效果图、施工计划、预算控制、材料与设备清单
- 设备采购：后厨设备、收银设备、监控与网络
- 试营业计划：菜单与价格、促销活动、人员安排、复盘清单

建议：每一步形成SOP与验收清单，减少返工。顺序建议：优先完成营业执照（至少受理/核发）后，再启动装修并支付租金，以降低风险。

## 营业执照申请（浙里办）—流程记录（2026-01-06）

- 状态：已通过浙里办发起申请（2026-01-06），等待审核/取照。
- 渠道：浙里办 App → 搜索“营业执照” → 进入“市场监管业务办理” → 选择“企业开办”。

### 操作步骤
- 打开浙里办 App，搜索“营业执照”。
- 依次进入“市场监管业务办理 → 企业开办”。
- 按流程指引，填写主体信息并上传资料后提交。
- 关注办件进度通知，按要求补正（如需）。
- 审核通过后，按指引线上领取电子营业执照或线下取照。

### 个体工商户资料准备清单
1. 执照名称（字号）
2. 经营者身份证正反面照片、手机号码
3. 经营范围（按实际经营项目选择/填写）
4. 房产证（每一页，含平面图页）、租赁合同（由经营者本人名义签署）

### 注意要点
- 经营范围尽量与后续食品经营许可范围匹配，避免二次变更。
- 租赁合同与房产证信息需一致，地址要规范到门牌号/楼层/铺位。
- 影像资料拍摄需清晰、无遮挡，确保所有页完整上传。
- 提交后及时关注补正短信/站内消息，避免因超时退回。

### 后续衔接
- 取得营业执照后：
	- 完成刻章与税务登记（如适用）。
	- 准备食品经营许可申报资料（平面图、设备布局、从业人员健康证明等）。
	- 在本仓“证照办理”节点更新状态与证照信息档案。

## 三小 vs 非三小：营业执照与许可流程区别

### 核心结论
- 营业执照（“一照一码，三证合一”）是市场主体登记，三小与否不改变“办照”流程本身；主要差异发生在办照后的食品许可环节。
- 三小通常指“食品小作坊、小餐饮、食品摊贩”。餐饮门店若规模较小、加工简单，可能按“小餐饮”办理备案；否则走“食品经营许可证（餐饮服务）”。

### 办照阶段（基本相同）
- 入口与步骤：浙里办 App → 企业开办 → 个体户/企业登记 → 填写主体信息与住所材料 → 提交受理 → 取得营业执照（统一社会信用代码）。
- 资料要点：名称、经营者身份证与手机号、经营范围、住所使用证明（房产证各页、租赁合同）。
- 差异仅在细节：经营范围表述建议与属地口径一致（如“餐饮服务”/“小餐饮”），避免后续许可变更；住所信息规范、与合同/产权一致。

### 证后分流（关键区别）
- 三小（小餐饮备案）：
	- 适用情形：面积较小、工艺简单、风险较低，由属地市监判定。
	- 办理形式：备案/登记管理，通常发放备案凭证/公示卡。
	- 关注点：平面布局与设备满足基本卫生要求；从业人员持健康证明；油烟与排污符合要求。
- 非三小（食品经营许可证·餐饮服务）：
	- 适用情形：规模/工艺/风险超过小餐饮范畴。
	- 许可要求：功能分区（粗加工、切配、烹饪、清洗消毒等）、通风排烟、冷藏冷冻、三防与清洗消毒设施、制度台账。

### 资料差异概览（证后环节）
- 小餐饮备案：平面图（含功能区）、设备清单、卫生制度与承诺、从业人员健康证明、场所合规说明等。
- 食品经营许可证：更完整的布局与流程文件、设备/设施配置证明、卫生与追溯制度、人员健康证明、可能的现场核查记录等。

### 建议流程
1. 先照：完成营业执照办理（已记录于上文“营业执照申请（浙里办）”）。
2. 咨询属地市监所：说明面积、座位数、菜单与设备，确认按“小餐饮备案”或“食品经营许可证”办理。
3. 按路径准备资料并预约/提交，现场核查通过后取证/备案。

提示：当前普遍实行“先照后证”。真正的把关点在证后许可阶段，装修设计应与拟办理路径的设施与功能分区要求一致，避免返工。

## 名词解释：食品经营许可证（餐饮服务）与“三证合一”

- 食品经营许可证（餐饮服务）：市场监管部门对餐饮服务单位开展食品经营活动的行政许可，适用于堂食/外卖餐饮等。核查重点包含功能分区、设施设备、卫生与流程控制、人员健康证明、管理制度与台账等。
- 三证合一（“一照一码”）：将原营业执照、组织机构代码证、税务登记证整合为一张营业执照，赋予统一社会信用代码，属于“市场主体登记”领域的证照改革。

关系与结论
- 不包含：个体工商户或企业的“三证合一”营业执照不包含“食品经营许可证”。两者分属不同事项，实行“先照后证”。
- 办理顺序：先通过浙里办完成营业执照登记（已在本文前文记录），再依据门店规模与菜单走“小餐饮备案”（三小）或“食品经营许可证（餐饮服务）”。
- 与三小关系：若属“小餐饮”，一般走备案/登记并取得备案凭证；若不属三小，则需办理“食品经营许可证（餐饮服务）”。无论备案还是许可，均独立于营业执照。

实操清单（证照路径示意）
- 一照一码：营业执照（统一社会信用代码）
- 其后其一：小餐饮备案（如判定为三小）/ 食品经营许可证（餐饮服务）
- 其他常见：公章刻制、税务事项办理、健康证、消防与环保类合规（按属地要求）

## FAQ

### 非三小，食品经营许可证需要单独办理吗？
答案：需要。三证合一（“一照一码”）仅整合营业执照、组织机构代码证、税务登记证，不包含“食品经营许可证（餐饮服务）”。应当先取得营业执照，再单独申请食品经营许可证。

办理概览（非三小·餐饮服务）
- 受理部门：属地市场监管部门（线上预约 + 线下核查为主）。
- 关键资料：
	- 场地与布局：平面图（含功能分区：粗加工、切配、烹饪、清洗消毒、仓储等）、动线与三防措施。
	- 设施设备：设备清单与配置（冷藏冷冻、通风排烟、清洗消毒、废弃物暂存等）。
	- 主体与人员：营业执照复印件、法定代表人/经营者信息、从业人员健康证明。
	- 管理制度：卫生与清洗消毒、原料采购与贮存、留样/追溯、培训与台账等。
- 关键节点：提交材料 → 现场核查 → 补正/整改（如需） → 核发许可证（纸质或电子证照）。
- 施工联动：装修与设备选型须满足许可要求，建议在设计定稿前与属地市监沟通版本要点，减少返工。
