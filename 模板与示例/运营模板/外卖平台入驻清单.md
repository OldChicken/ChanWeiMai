# 外卖平台入驻自查清单（美团 / 京东到家 / 饿了么）

- 资质与证照
  - 营业执照、食品经营许可证、法人/负责人身份证
  - 银行账户信息（结算）、员工健康证
- 门店信息
  - 店铺名、简介、地址、营业时间、休息日
  - 门头/后厨/环境照片合规（清晰、无遮挡）
- 配送设置
  - 平台骑手/自配送选择、配送范围、配送费与时效
  - 起送价与包装费设置
- 菜单与素材
  - 类目、SKU、规格、图片、文案、标签（辣度/招牌/新品）
  - 套餐置顶与引流SKU设置
- 活动与折扣
  - 满减梯度、限时折扣、拉新券、评价返券
  - 预算与折扣占比控制
- 测试与演练
  - 自测试单流程（下单→出餐→配送→收货→评价/退款）
  - 异常演练（超时/取消/错误出餐）
